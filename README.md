# GoLang-TaskManager
# Requirements

- Go
- SQLite (no setup required the database file is created automatically)

# Running the Project

1. Clone the repository:
    ```
    git clone https://github.com/Aidanu504/GoLang-TaskManager
    cd golang-taskmanager
    ```

2. Run the server:
    ```
    go run main.go
    ```

3. Open the application in a browser at:
    ```
    http://localhost:8080
    ```
